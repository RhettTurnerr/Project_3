## Description:
This is a website that has a Todo page and a Contact page. In the Todo page the user can input text that will be displayed below. The user can then mark these tasks as complete, incomplete, or delete them. In the Contact page the user is prompted to enter their name and email as well as any comments about the website.

## User Stories
* The user will click todos page to navigate to todos page. Then they will input a task into the text input and click add. Their task will then appear below. They will then click the contact page to navigate to the contact page. They will then input their name, email, and comments about the website. Then they will click submit.
* The user will click the todos page to navigate to the todos page. They will input a task into the text input and click add. They will then click the box next to the task to mark it complete. 
* The user will click the contact page to navigate to the contact page. They will then input their name, email, and comments about the page. They will then click submit. They will then click todos page to navigate to todos page. They will then enter a task and click add. They will enter another task and it will appear below the first one. The user will then click the box next to the first task marking it complete.
